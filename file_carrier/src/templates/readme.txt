--------------------------------------------------------------

              TRANSPORTEUR DE DONNÉES ARCHIPEL

--------------------------------------------------------------

Archipel est un réseau décentralisé de transport de données
qui se base (entre autres) sur l'échange de clés USB. Cette clé 
USB permet le transport de données pour vous ou pour d'autres
personnes.

Les données sont stockés sur la clés et bougent de clés en clés
grâce aux personnes qui s'y connectent.

Vous aurez besoin du logiciel "Archipel Core" et "Archipel file
carrier" pour faire partie du réseau. Rendez vous sur le site
suivant pour en savoir plus.

https://github.com/EpicKiwi/archipel-core

--------------------------------------------------------------

                    ARCHIPEL FILE CARRIER

--------------------------------------------------------------

Archipel is a decentralized data transport network which is
based (among others) on the exchange of USB keys. This USB key
allows the transport of data for you or for other people.

The data are stored on the key and move from key to key
thanks to the people who connect to it.

You will need the "Archipel Core" and "Archipel file
carrier" to be part of the network. Go to the following site
to learn more about it.

https://github.com/EpicKiwi/archipel-core
